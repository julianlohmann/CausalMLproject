## Dataset Information

### Dataset Source
- **Dataset Link:** https://www.oecd.org/en/data/datasets/pisa-2022-database.html

### Dataset Characteristics
- **Number of Observations in the German PISA 2022 Sample:** ca. 6100
- **Number of Features:** >100

### Target Outcome Variable
- **Name:** PISA Reading Competence 
- **Type:** Continuous (Mean = 500; SD = 100)

### Treatment / Exposer Variable
- **Name:** Pre-primary school education
- **Type:** Dichotomous (0 = no pre-primary school, 1 = pre-primary school)

### Core covariates
- HISEI
- Gender
- resources at home
- parental characteristics
- ESCS
- HISCED
- IMMIG
- (...)
